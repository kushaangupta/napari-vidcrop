# napari-vidcrop

Useful video cropping operations

----------------------------------

### Setup

```bash
pip install .
```

### Usage
Load the plugin from Plugins menu inside the napari window.

---

## For development
### Setup

```bash
pip install poetry
poetry install
```

### Usage

```bash
poetry shell
pip install -e .
napari
```

Then load the plugin from Plugins menu inside the napari window.
